Game starts by simply clicking on Box or by using the num keys.First player as X & Second player as 0












Submitted by : Bishal Nandi

Created a Tic-Tac-Toe game in Vanilla JavaScript where 2 players can play and put values in the boxes.. either using mouse or buttons.

Additional Features added ➕ :

  • ✅ The final result will be stored in the local storage.
  • ✅ When the user re-opens or comes back to play game, it will first show him/her the past match leader board and then he/she can play current match
  • ✅ The game is automated by making second player as computer which can play with 1 player.
